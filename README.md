# Knight1
Game created using GODOT, using gdscript as the primary language. It has a knight, preused pallet template from itchi.io.
The game is still in its beginning phase and is made completely from tutorial. 

This game consist of a Knight(Playable Character), Slime(enemy), coins and platform.

Kngiht consist of 3 different frames Idle, Run and Jump. Except that it also has a death animtion when the user encounter the enemy or falls off the plaform.

Slime only has 1 animation that is idle he goes from right to left changing direction.

Coins are the in game currency

Platform are of 2 types.
1)Still: User stands on this platform and runs around it. These are the stationary objects.
2)Moving: These are used to travel from one place to another.
